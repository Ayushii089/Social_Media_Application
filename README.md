# Social_Media_Application
A full-stack Social Media Application built using HTML, CSS, JavaScript, Bootstrap, React.js, Node.js, Express.js, and MongoDB.
The app supports user authentication, post creation, and like functionality, following a modern MERN stack architecture with a responsive UI and RESTful APIs.

Features:

User Registration & Login (JWT Authentication)

Create and View Posts

Like Posts

Responsive UI with Bootstrap

MongoDB Database Integration

Tech Stack:
React | Node.js | Express.js | MongoDB | Bootstrap | JavaScript
